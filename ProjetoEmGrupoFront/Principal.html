Vamos falar sobre CSS



O que é CSS?

O HTML é muito estático, por isso usamos o CSS (Cascading Style Sheets), que é uma linguagem de estruturação que está voltada a página na Web, geralmente aplicada ao HTML, mas não confunda, CSS não é uma linguagem de programação.



Conceito

O CSS (Cascading Style Sheets) é a linguagem que estrutura que está voltada a página na Web, geralmente aplicada ao HTML, mas não confunda, CSS não é uma linguagem de programação.

O CSS descreve como os elementos HTML devem ser apresentados na tela

O código de estilo pode ser reaproveitado por múltiplas páginas. O arquivo de código de estilos é do tipo “.css”

O CSS usa regras e propriedades para customizar elementos.

História

O HTML não surgiu com intuito de conter TAGs para formatar páginas da internet. Com a complexidade das páginas web, o processo de manutenção ficou difícil.

A W3C criou então o CSS, que remove a formatação das TAGs HTML.
